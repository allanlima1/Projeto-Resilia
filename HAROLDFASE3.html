Harold sai de floresta, e derrepente se dá conta de que está no castelo de Java, o vilão de resilia. Forte e destemido ele invade o castelo e chega a sala do trono. Java o esperava e travaram uma batalha épica.
Quando Java se dá conta do poder de Harold, ele decide que a melhor maneira de acabar com ele seria explodindo o castelo com um temido feitiço lendário. Harold se dá conta dos planos de Java e deve tomar uma decisão.
