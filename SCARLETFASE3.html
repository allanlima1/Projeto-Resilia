Scarlet sai de floresta, e derrepente se dá conta de que está no castelo de Java, o vilão de resilia. Forte e destemido ele invade o castelo e chega a sala do trono. Java a esperava e os dois travaram uma batalha épica.
Quando Java se dá conta do poder de Scarlet, ele decide que a melhor maneira de acabar com ela seria explodindo o castelo com um temido feitiço lendário. Scarlet se dá conta dos planos de Java e deve tomar uma decisão.
